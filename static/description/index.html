Módulo VeriFactu para AEAT (Odoo 16)

Este módulo permite integrar el sistema VeriFactu de la Agencia Tributaria (AEAT) con Odoo 16, cumpliendo con la normativa de envío de registros de facturación de forma automática y segura.

Características

  • Genera el hash de cada factura según las especificaciones de la AEAT
  • Genera el QR requerido para el control de la factura
  • Crea el XML conforme al esquema oficial de VeriFactu
  • Permite el envío automatizado de las facturas a la AEAT
  • Almacena y muestra el estado de envío de cada factura
  • Bloquea modificaciones en facturas ya enviadas a la AEAT
  • Compatible con Odoo 16.0

Instalación

  1. Descargue e instale el módulo como cualquier otro módulo de Odoo
  2. Reinicie el servidor de Odoo
  3. Actualice la lista de módulos
  4. Instale el módulo desde el backend de Odoo

Configuración

Tras la instalación, configure los datos necesarios del certificado digital y los parámetros de conexión con la AEAT desde el menú de ajustes técnicos.

Nota para desarrolladores

El módulo implementa el envío mediante SOAP/XML firmado y validado según las directrices de la AEAT. El código es fácilmente ampliable para otras versiones o integraciones.
